# Mandelbrot Set
A Mandelbrot Set generator.

### Built With
* JavaScript
* HTML 5 Canvas

## Demo
Check the live demo here: https://patrickmargot.github.io/mandelbrot-set/

## Installation
1. Clone repository
```sh
git clone https://github.com/PatrickMargot/mandelbrot-set.git
```
2. Open `index.html` in your browser
